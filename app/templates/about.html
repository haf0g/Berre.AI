About Berre.AI

Berre.AI is an innovative chatbot solution powered by artificial intelligence, specifically designed to assist visitors and locals in discovering Berrechid. Our solution combines the power of Groq's LLM technology with a sophisticated RAG (Retrieval-Augmented Generation) system to provide accurate, contextual information about the city.

Built using cutting-edge technologies:
  • Groq API for natural language processing
  • RAG system for accurate local information retrieval
  • Flask backend for robust server handling
  • Responsive frontend for seamless user experience
